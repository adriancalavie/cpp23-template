# C++23 Template

An opinionated C++23 project template using [xmake](https://xmake.io) with a batteries-included developer experience — clangd, clang-tidy, clang-format and custom tasks out of the box.

## Requirements

- [xmake](https://xmake.io/guide/quick-start.html)
- [LLVM](https://llvm.org/) via Homebrew (`brew install llvm`) — required for full C++23 stdlib support on macOS

## Getting started

```bash
git clone <repo>
cd cpp23-template
xmake
xmake run
```

## Tasks

This template ships with three custom tasks on top of the standard xmake commands.

| task | description |
|---|---|
| `xmake debug` | configure, build and run in debug mode |
| `xmake release` | configure, build and run in release mode |
| `xmake clear` | wipe `build/` and `.xmake/` cache |

All tasks support `--help`:

```bash
xmake debug --help
xmake clear --help
```

`xmake clear` accepts an optional flag to also remove `compile_commands.json`:

```bash
xmake clear --cc
xmake clear -c   # shorthand
```

## Project structure

```
.
├── src/
│   └── main.cpp
├── .clangd              # clangd config (LLVM stdlib path, flags)
├── .clang-format        # formatting rules (snake_case, left pointers, etc.)
├── .clang-tidy          # linting rules (naming conventions, bugprone, modernize)
├── .luarc.json          # Lua LSP config for xmake globals
├── compile_commands.json# auto-generated by xmake, used by clangd
└── xmake.lua            # build config + custom tasks
```

## Code style

Naming follows Rust conventions:

| construct | convention |
|---|---|
| types, structs, enums | `PascalCase` |
| functions, variables, parameters | `snake_case` |
| constants, macros | `UPPER_SNAKE_CASE` |
| private members | `snake_case_` (trailing underscore) |

Pointer and reference alignment is left: `int* x`, `int& y`.

See [.clang-format](./.clang-format) and [.clang-tidy](./.clang-tidy) for the full config.

## Debug macros

xmake defines `NDEBUG` in release mode and `_DEBUG` in debug mode (standard C++ convention). Use them like:

```cpp
#ifdef NDEBUG
    std::println("release build");
#else
    std::println("debug build");
#endif
```
